我是谁

爱折腾，伪文青，斯人已去，勿念，现在学习编程。(2016-2018）

haigeek是我在网络上常用的昵称，因为他的谐音和我们真实名字很像，当然里面还包含了geek这个单词，这个名字是在大二在一节无聊的数学课上起的，当然我并不认为是我是一个真正的极客，但是可以来代表一直偏执的精神。

我这人也是这样的，没有一个真是符合自己的定义，我也没办法完全认清自己，我在努力做好一些维持生活的事情，比如赚钱工作可以让自己在世界存活下去。温饱之余，总是想去尝试一些没做过的事情，很多时候都是半吊子的选手，这些个过程中我觉得是和自己的一次对话。

我有几个和我一起度过大学时光的朋友，每次和他们深层次聊天的时候，我发现会有了一些新的经历，有了一些新的想法，也许是和以前相悖的，为此我并不为之感到奇怪。

虽然这世界有太多让人类觉得不可思议或者叹为观止的事情，我始终相信在有限的生命里要做自己想做的事情，难免会犯错，但是年轻的时候是不是要多试错呢？

我的折腾

编程

关于编程我一直也路上，也许有一天我会发现我不适合从事程序开发，但是我觉得就算我不能把这件事做的很好，我依旧要认真对待，人之所以是高级动物是因为他有思考的能力和追求成果的信念，这个准则在任何领域应该都是适用的。

  • java 是我现在吃饭的语言，但是我也只是停留在很浅的一个层面
  • 大学的时候学了c++、c#但是本质上我并不能说我会这两门语言
  • python是我毕设使用的语言，这是一门开箱即用并具有魔力的语言
